# Local Supabase Bootstrap

This directory holds local-Supabase testing artifacts for sync-enabled durability work.

Expected files:

- `test.env`
  - Generated by `./scripts/local-supabase.sh provision-users`
  - Contains local URL/key values and deterministic test-user credentials
  - Treated as generated/local state and not required to be committed

The local database schema comes from `supabase/migrations/` (a single baseline
migration kept in sync with `spread-prod`'s schema). `./scripts/local-supabase.sh reset`
runs `supabase db reset`, which replays those migrations against the local database —
no separate bootstrap/dump step is required.
